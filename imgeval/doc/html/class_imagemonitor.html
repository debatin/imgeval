Detailed Description

Class for displaying the images on the screen.

Stores the data in an Imagedata class. And displays it using different colorscheemes. Also zooming capabilities and regions of interes should be implemented here.

void Imagemonitor::createPalette (  )

creates the palette from a vector of colors

creates the Palette vrom a vector of colors stored in the central settings The vector should have n*3 entries which are interpreted as R1 G1 B1 R2 G2..... where R1 is the red component of the first color. The colors are placed equidistant in the number space and a smooth transition between the colors is created.
